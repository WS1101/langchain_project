# 요구사항

사용자의 투자 등급(기관/개인)과 리밸런싱 요청 금액을 입력받아, 등급별 자산 보유 한도 초과 여부를 계산하고 고액 거래 시 필요한 추가 인증(MFA) 절차를 안내하는 로직을 구현하시오.

시장 급락 상황(ID 9: 기술주 하락 시나리오)이 감지되었을 때, 해당 현상의 근거가 되는 거시경제 지표(ID 10: 장단기 금리 역전)를 지식 베이스에서 찾아 연결하고, 이로 인해 에이전트가 수행하게 될 다음 분석 단계(ID 11: 고용 지표 확인)를 설명하는 기능을 구현하시오.

사용자가 해외 주식이나 채권 매수를 요청할 때, ID 26(환율 변동성), ID 30(한미 금리차), ID 39(투자 적합성) 데이터를 순차적으로 검증하시오.
